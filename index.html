Quickstart Guide

See how easy it is to get started.

1. Connection & Setup

Instantiate `SQLBridge` once in your application. It automatically manages a high-performance connection pool.

from kairondb import SQLBridge, Model, Q

# This creates a persistent connection pool
bridge = SQLBridge(
    driver="sqlserver",
    server="YOUR_SERVER_IP",
    db_name="YOUR_DATABASE",
    user="YOUR_USERNAME",
    password="YOUR_PASSWORD"
)

# Create a model for your table
user_model = Model(bridge, "Users")

2. Performing Queries with Lookups

Use double-underscore lookups in the `where` dictionary for powerful filtering.

# Find users older than 30
older_users = user_model.select(where={'age__gt': 30})

# Find users whose name contains 'Maria'
maria_users = user_model.select(where={'name__like': '%Maria%'})

# Find users with 'active' or 'pending' status
active_users = user_model.select(where={'status__in': ['active', 'pending']})

# Update records matching a condition
user_model.update(data={'status': 'archived'}, where={'last_login__lt': '2023-01-01'})

Advanced API

Combine filters and manage data integrity.

Complex Queries with Q Objects

For `OR` conditions or complex logic, import and use `Q` objects. Combine them with `&` (AND) and `|` (OR).

# Find users who are older than 50 OR have an 'admin' status
privileged_users = user_model.select(
    where=Q(age__gt=50) | Q(status='admin')
)

# Find active users AND (whose name starts with 'J' OR are younger than 25)
complex_filter = user_model.select(
    where=Q(status='active') & (Q(name__like='J%') | Q(age__lt=25))
)

Atomic Transactions

Use the `transaction` context manager to ensure a group of operations all succeed or all fail together.

try:
    with bridge.transaction() as tx:
        # 'tx' is a special bridge that only works inside the transaction
        accounts_model = Model(tx, "Accounts")
        
        # All operations here are atomic
        accounts_model.update(data={'balance': 900}, where={'id': 1})
        accounts_model.update(data={'balance': 1100}, where={'id': 2})
        
    print("Transfer completed successfully!")
except Exception as e:
    print(f"The transaction failed and was rolled back: {e}")
