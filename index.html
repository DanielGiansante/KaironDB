Why KaironDB?

A modern approach to classic data access problems.

Unmatched Performance

Using a Go backend, KaironDB minimizes Python's overhead to deliver drastically faster query executions.

Intuitive Syntax

With an ORM-like API, you can perform complex operations with simple, direct methods like `.select()` and `.insert()`.

Multi-DB & Transaction Support

Connect to SQL Server, MySQL, and PostgreSQL without changing your code. Ensure data integrity with safe and easy-to-use transactions.

Quickstart Guide

See how easy it is to get started.

1. Installation

Install the library using pip with the provided .whl file.

pip install kairondb-0.1.0-py3-none-any.whl

2. Connection

Create an `SQLBridge` instance, passing your credentials directly.

from kairondb import SQLBridge, Model

# Create the connection bridge
bridge = SQLBridge(
    driver="sqlserver",
    server="YOUR_SERVER_IP",
    db_name="YOUR_DATABASE",
    user="YOUR_USERNAME",
    password="YOUR_PASSWORD"
)

# Create a model for your table
user_model = Model(bridge, "Users")

3. Performing Operations

Use the model's intuitive methods to interact with the database.

# Fetching all users
all_users = user_model.select()
print(all_users)

# Fetching a specific user
user = user_model.select(where={'id': 1})
print(user)

# Inserting a new record
new_user = {'name': 'Ana', 'email': 'ana@example.com'}
result = user_model.insert(new_user)
print(result) # Output: {'rows_affected': 1}

Using Transactions

Ensure data integrity with the `transaction` context manager. Operations either all succeed (commit) or all fail (rollback).

try:
    with bridge.transaction() as tx:
        # 'tx' is a special bridge that only works inside the transaction
        accounts_model = Model(tx, "Accounts")
        
        # All operations here are atomic
        accounts_model.exec("UPDATE Accounts SET balance = balance - 100 WHERE id = 1")
        accounts_model.exec("UPDATE Accounts SET balance = balance + 100 WHERE id = 2")
        
    print("Transfer completed successfully!")
except Exception as e:
    print(f"The transaction failed and was rolled back: {e}")
